Embed content from a whole list of sites that don't natively support oEmbed. Uses embeds from oembed.io and noembed.com.

== Description ==

Noembed allows you to embed content from a large list of sites that don't natively support oEmbed. Embeds are provided by [oembed.io](http://oembed.io/) and [noembed.com](http://noembed.com/).

== Installation ==

 * Upload and install this plugin as normal, and activate it from the Plugins menu in WordPress.
 * Insert a URL from one of the supported sites into your post content and then view the post.
 * Bam! The content will be embedded.

For an up-to-date list of supported sites, please visit [oembed.io](http://oembed.io/) and [noembed.com](http://noembed.com/).
